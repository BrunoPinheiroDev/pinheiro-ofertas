\# 🌲 Pinheiro Ofertas - WordPress Theme



!\[GitHub language count](https://img.shields.io/github/languages/count/BrunoPinheiroDev/pinheiro-ofertas?style=for-the-badge)

!\[GitHub top language](https://img.shields.io/github/languages/top/BrunoPinheiroDev/pinheiro-ofertas?style=for-the-badge\&color=green)



Este é um tema customizado para WordPress desenvolvido para o projeto \*\*Pinheiro Ofertas\*\*, um portal focado em curadoria de ofertas de tecnologia e eletrônicos.



\## 🚀 Tecnologias Utilizadas

\- \*\*PHP:\*\* Lógica do WordPress e integração com Banco de Dados.

\- \*\*CSS3:\*\* Estilização customizada e responsiva.

\- \*\*JavaScript:\*\* Interações de interface.

\- \*\*WordPress CMS:\*\* Estrutura base do projeto.



\## ✨ Funcionalidades

\- \[x] Listagem de ofertas com filtros customizados.

\- \[x] Design focado em conversão de cliques (Affiliate Marketing).

\- \[x] Integração com ícone de WhatsApp para suporte/consultoria.

\- \[x] Carregamento otimizado de imagens e scripts.



\## 🛠️ Como rodar o projeto

1\. Tenha um ambiente WordPress instalado (ex: Local WP).

2\. Clone este repositório dentro da pasta `/wp-content/themes/`.

3\. Ative o tema no painel administrativo do WordPress.
